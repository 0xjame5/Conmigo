# Cheesy P2P

Multi-rooms WebRTC P2P messaging web app, containing a light server written in Python that provides web services for AngularJS module.

## Tools

- **Python** with *Flask*
- **WebRTC**
- **AngularJS**

## Requirements

- Python >= 2.7
- Browser support for WebRTC
